# Fluff

Yu-Gi-Oh Health Tracker written in Flutter

## Features
* track your and your enemies health
* log history for each game
* portrait & landscape mode
* delete games or continue older Games

## Dependencies
* path_provider: ^2.1.4
* uuid: ^3.0.6
